Detailed Description

This class allows users to select which body redirection technique to use as well as the relevant parameters. The enumeration BRtechnique allows for dynamic selection of the technique. The class reference BodyRedirectionTechnique contains the redirect function that is used to apply redirection with the Redirect() function. The scene parameter contains all the information necessary for the redirection such as the GameObjects representing the hand (physical and virtual) or the targets.
