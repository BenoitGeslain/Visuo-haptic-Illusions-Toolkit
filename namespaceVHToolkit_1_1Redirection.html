record VHToolkit.Redirection.Scene ( )

This class records the position of various objects of interest.

Parameters
physicalHandThe Transform representing the user's physical hand, i.e. the transform that is receiving the tracked hand data.
virtualHandThe Transform representing the user's virtual hand, i.e. the user's avatar.
physicalHeadThe Transform representing the user's physical head, i.e. the transform that is receiving the tracked head data..
virtualHeadThe Transform representing the user's virtual head.
Parameters
physicalTargetThe physical target the hand should reach when the virtual hand reaches the virtual target.
virtualTargetThe virtual target the user is reaching for.
originThe point of origin where no redirection is applied.
targetsThe target(s) used for the steering strategies. For SteerToOrbit, the first element is considered to be the center of the orbit.
radiusThe radius of the orbit in the SteerToOrbit strategy.
applyDampeningWhether to apply dampening in the Razzaque et al.'s Redirected walking hybrid technique.
applySmoothingWhether to apply smoothing in the Razzaque et al.'s Redirected walking hybrid technique.

The position of the virtual limb is given by physicalHand.position + Redirection.
